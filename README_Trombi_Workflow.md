# Workflow "Zéro Risque" : Shooting Photo Trombinoscope (1000 personnes)

Ce dossier contient tout ce dont vous avez besoin pour gérer le shooting d'Europ Assistance sans aucune erreur de renommage.

## 🎯 Le Fichier Programme
Le programme générateur de QR Code est autonome et prêt à être envoyé sur l'iPad du client :
**Fichier :** `index.html` (à ouvrir avec Safari, fonctionne sans internet).

## 📋 La Méthode "Visuelle" Infaillible
Laissez la donnée suivre la photo physiquement :

1. **Préparation de la Liste :** Le client a son fichier RH "Nom, Prénom".
2. **Accueil & iPad :** L'assistant accueille l'employé, tape ses coordonnées et génère le Badge Plein Écran `Prénom NOM_date`.
3. **Photo de Sécurité (La 1ère) :** L'employé tient l'iPad (légèrement incliné vers le sol pour éviter le reflet du flash). Vous photographiez la personne avec le QR Code.
4. **Photos de Portrait :** Vous prenez les vraies séries de portraits sans l'iPad.
5. **Post-Production (Magie) :** Vous passez les photos dans votre logiciel (GotPhoto, Lightroom + Plugin QRSnip/Tethered Barcode). Le logiciel voit un QR code, il lit "Jean DUPONT_12042026", et il renomme automatiquement **toutes** les photos qui suivent avec ce nom, jusqu'au prochain QR code. Le décalage est mathématiquement impossible.
